# Book Summary App

A React application for storing and managing book summaries.

## Features

- View book summaries
- Add new book summaries
- Browse your library
- User login and registration pages
- Responsive navigation bar

## Technologies

- React
- Vite
- React Router DOM
- CSS

## Project Structure

```txt
src/
├── App/
├── components/
├── pages/
│   ├── Home/
│   ├── Library/
│   ├── Login/
│   └── Register/
├── main.jsx
└── index.css
```

## Installation

```bash
npm install
```

## Run the Project

```bash
npm run dev
```
